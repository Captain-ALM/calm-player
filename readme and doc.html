Introduction


This documentaition will show you how to use the scripting langauge.
This scripting can be used in the data file in the data folder/archive.
You pack the folder into a zip archive (You can do this within the program by opening the folder up with the program and then pressing shift+F2).

File Format


You must use the .txt file format for data files in the data format.
Code in either notepad or notepad++.
You must use the .jpg,.gif,.png formats for the pictures.
You must use the .wav,.mp3,.wma,.ogg formats for sound.
You must use the .wav,.mp3,.wma,.ogg,.ogv,.avi,.wmv,.mp4,.mpeg,.mpg,.mpeg-4 formats for media.

Scripting Syntax


This is the synatx for the data files:
An object name.
Then a ':'
Then the parameters required for the object after the : spaced by a comma.
Then a ';'
Eg: location:5,5;
Return to Documentaition Contents


Inroduction to Data File Scripting


Quiz file system involves using the Scripting Syntax to code the data files.
These data files are in .txt format and can be named anything.
On the first line, put in the page number of the data file.
On the next lines you can put the code, each object section is defined on a different line.
Eg:
1
type:static_image;location:5,5;data:splash.png,500,239;show_animation:1000,1,true;hide_animation:1000,1,true;
type:text;data:hello world,Green,Arial,36,!;show_animation:1000,5,false;hide_animation:1000,1,true;location:100,100;

Type


type:aig/animated_image/img/image/static_image/tit/title/txt/text/sound/music/audio/media/video/motion_picture/run/background_color/background_colour/background_image;
Type of object.
aig/animated_image = animated image, img/image/static_image = static image, tit/title = title bar text, txt/text = text, run = run program, sound/music/audio = play sound, media/video/motion_picture = play media file, background_color/background_colour = background colour, background_image = background image.

Show Animation


anims/show_animation:time(milliseconds),from(number),true/false(fade in/out);
The show animation of the object.
The from number list: 0 = none, 1 = top center, 2 = top right, 3 = center right, 4 = bottom right, 5 = bottom center, 6 = bottom left, 7 = center left, 8 = top left.
Return to Documentaition Contents


Hide Animation


animh/hide_animation:time(milliseconds),from(number),true/false(fade in/out);
The hide animation of the object.
The from number list: 0 = none, 1 = top center, 2 = top right, 3 = center right, 4 = bottom right, 5 = bottom center, 6 = bottom left, 7 = center left, 8 = top left.
